# cockpit for pixiv
Provide comfortable pixiv browsing.

## Features and Usage
visit [pages](https://8th713.github.io/cockpit-for-pixiv/).

## Development
```sh
npm install
```

To watch and development build:
```sh
npm start
```

To production build:
```sh
npm run build
```

To type check:
```sh
npm run tsc
```
