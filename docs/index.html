cockpit for pixiv

pixiv の閲覧がより快適になるユーザースクリプト

Install cockpit for pixiv

Features

  • 画面遷移なしでイラスト表示、複数枚投稿・うごイラ対応
  • イラストの画面フィット表示・原寸表示、Grab and Drop によるスクロール対応
  • その場でブックマーク、10点評価
  • サイドパネルでキャプションやタグを表示
  • ショートカットキー操作可能
  • アドオン連携で画像のローカル保存に対応(cockpit download addon)

Install

cockpit for pixiv は Google Chrome + Tampermonkey 環境で実行される UserScript です。

準備

  1. Google Chrome をインストールします。
  2. Tampermonkey をインストールします。
  3. このページのインストールボタンから「cockpit for pixiv」をインストールします。
  4. ローカル保存機能を使用する場合「cockpit download addon」をインストールします。

インストール

Install [cockpit for pixiv - v1.0.1]

Install [cockpit download addon]

注意

  • スマートフォン、Mozilla Firefox + Greasemonkey 環境での動作確認は行っていません。
  • pixiv は、ピクシブ株式会社の登録商標です。
  • cockpit for pixiv は pixiv が作成、配布しているアプリケーションではありません。
