cockpit for pixiv

pixiv の閲覧がより快適になるユーザースクリプト

Install cockpit for pixiv

Features

漫画表示・うごイラにも対応

通常のイラスト表示に加え、漫画も1ページずつ表示することができます。さらに、うごイラの再生にも対応。うごイラのフルスクリーン表示もできます。

via (pixiv事務局 | うごイラはじめました)

イラストの画面フィット表示・原寸表示もワンアクションで切替可能

オリジナル画像を画面内に収まるようにリサイズして表示できます。その場ですぐに原寸に切り替えることが可能です。

via (pixiv事務局 | 【コミックマーケット86】pixivブースお品書き)

漫画のサムネイル一覧表示

漫画表示中、作品のサムネイル一覧を表示して気になるページにすぐジャンプできます。

via (瀬尾 | 風景イラストまとめ)

その場でブックマーク

ページを移動することなくその場で作品をブックマークしたり作者をフォローすることが可能です。評価やアンケートへの回答にも対応しています。

via (pixiv事務局 | うごイラはじめました)

コメントもその場で表示

作品へのコメントをその場で読むことができます。スタンプや絵文字の表示にも対応しています。

via (pixiv事務局 | うごイラはじめました)

Install

cockpit for pixiv は Google Chrome + Tampermonkey 環境で実行される UserScript です。

一部機能を除き Firefox + Greasemonkey でも動作します。

準備

  1. Google Chrome をインストールします。
  2. Tampermonkey をインストールします。　

インストール

Install [cockpit for pixiv - v0.1.5]

注意

  • Mozilla Firefox + Greasemonkey 環境での動作確認は行っていません。
  • pixiv は、ピクシブ株式会社の登録商標です。
  • cockpit for pixiv は pixiv が作成、配布しているアプリケーションではありません。
